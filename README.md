# Bloc de Notas

Este proyecto en **Java** implementa tres interfaces gráficas diferentes relacionadas con el registro y login de una página web:

- **Interfaz sencilla:** Permite el registro y acceso de usuarios de forma básica.
- **Interfaz compleja:** Ofrece opciones adicionales y una experiencia más avanzada para el registro y el login.
- **Interfaz combinada:** Integra elementos de las dos anteriores, logrando flexibilidad y facilidad de uso.

La aplicación está pensada para demostrar distintas formas de gestionar procesos de autenticación y registro de usuarios en Java, utilizando interfaces gráficas (por ejemplo, Java Swing).

## Instalación y ejecución

1. Clona el repositorio:
   ```bash
   git clone https://github.com/BeNiTez-prog/Bloc_de_Notas.git
   ```
2. Compila los archivos Java:
   ```bash
   javac *.java
   ```
3. Ejecuta la interfaz que desees probar:
   ```bash
   java NombreDeLaClasePrincipal
   ```
   (Reemplaza `NombreDeLaClasePrincipal` por el nombre de la clase que inicia la interfaz)
